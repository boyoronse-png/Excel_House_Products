# Excel_House_Products
Using LOOKUP techniques to analyse product, pricing, supplier, discount, and shipping data in a business context.

This project demonstrates my ability to use Excel lookup functions to retrieve and analyse product, supplier, pricing, stock, and order data.

## Skills demonstrated
- XLOOKUP
- Exact match lookup
- Approximate match lookup
- Wildcard lookup
- Multi-column return
- Data validation and checking
- Spreadsheet organisation

## Project overview
I used a product dataset to answer lookup-based business questions, including product identification, price retrieval, supplier information, discount lookup, shipping fee lookup, and missing SKU handling.

## Tools used
- Microsoft Excel
- XLOOKUP
- Structured data tables
